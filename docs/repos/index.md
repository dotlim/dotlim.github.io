## Moa

计划设计一款基于 Webpack5 & Module Federation 实现的跨应用共享模块的工具，旨在减少微前端应用之间模块共用等场景。

现在市面上 single-spa, quankun 等微前端方案大方向是解决不同框架之间的关联，而 Moa 更致力于实现模块共享价值，粒度小到模块。
